# Kitty Kollector

A 2D pixel‑art cat platformer made for the Cat Game Jam.  
In the city of Rickleback, Nana Rose runs a safe haven for stray kitties on 7th Street—but she’s too old to go out and save them herself. Now it’s up to **Trixie**, a once‑stray cat, to rescue kittens, find refreshments, and dodge pest‑control patrols.

## Features
- Cute 2D pixel‑art cat hero and kittens.  
- Small, hand‑crafted platforming levels.  
- Simple run, jump, hide, and dash mechanics.  
- Designed to be family‑friendly and cat‑positive.  

Made solo with Unity.
